[![Build status](https://api.travis-ci.org/Angular2Buch/code.svg)](https://travis-ci.org/Angular2Buch/code)

## Demo-Seite starten

```cmd
$ npm install
$ npm install -g jspm
$ jspm install
$ npm run-script build
$ npm start
```
